💻 系统要求

核心依赖

📦 libvips 图像处理库

识文君依赖 libvips 库进行PDF页面渲染和图像处理。这是软件正常运行的必需依赖。

不同平台的安装方式

🍎 macOS

安装方法：

brew install vips

支持 Intel 和 Apple Silicon 芯片

注意：首次运行前请确保已安装 libvips

🪟 Windows

安装方法：从 GitHub 下载预编译包

https://github.com/libvips/build-win64-mxe/releases

下载 vips-dev-w64-all-8.12.2.zip 并解压到系统路径

支持 Windows 10/11 x64

注意：首次运行前请确保已正确配置 libvips

🐧 Linux

Ubuntu/Debian：

sudo apt-get install libvips-dev

CentOS/RHEL：

sudo yum install vips-devel

Fedora：

sudo dnf install vips-devel

Arch：

sudo pacman -S libvips

依赖检查方法：

  1. 启动识文君应用
  2. 软件会自动检测 libvips 是否可用
  3. 如果依赖缺失，会在界面上显示相应提示
  4. 按照提示安装对应的依赖包

⚠️ 重要说明

必需依赖：识文君需要 libvips 库才能正常运行，请在首次使用前安装。
版本要求：建议使用 libvips 8.12+ 版本以获得最佳兼容性。
自动检测：软件启动时会自动检测依赖是否可用，如有缺失会显示提示。

📖 详细使用指南

1. 加载文档

  1. 点击界面中的"选择文件"按钮
  2. 选择PDF文件（目前仅支持PDF格式）
  3. 等待文档加载完成
  4. 在左侧预览区查看文档内容

📋 支持格式说明

当前版本仅支持PDF文档识别处理，图片格式支持正在开发中，敬请期待后续版本更新。

2. 文字识别

  1. 选择要处理的页面（可单页或多页）
  2. 点击"开始处理"按钮
  3. 观察处理进度，等待识别完成
  4. 在右侧查看识别结果

3. 编辑文本

  1. 点击页面右侧的"编辑文本"按钮
  2. 在弹出的编辑器中修改识别结果
  3. 支持拖拽调整编辑器位置和大小
  4. 编辑完成后点击"保存"

4. AI文本处理

  1. 在识别结果区域点击"AI处理"按钮
  2. AI会自动优化文本格式和内容
  3. 处理结果支持Markdown渲染显示
  4. 可以重新处理或编辑AI处理结果

📤 导出功能

支持的导出格式

📄 TXT文本

纯文本格式，兼容性最好，适合简单的文本内容。

📝 Markdown

支持格式化的文本，保留标题、列表、表格等结构。

📋 Word文档

DOCX格式，支持复杂排版，适合正式文档。

🌐 HTML网页

网页格式，支持富文本显示，便于在线查看。

📑 RTF富文本

富文本格式，兼容多种文字处理软件。

单页导出：

  1. 在页面右侧点击"导出"按钮
  2. 选择导出格式（TXT、Markdown、Word、HTML、RTF）
  3. 选择导出内容（OCR文本或AI处理文本）
  4. 点击"导出"完成

批量导出：

  1. 点击顶部工具栏的"历史记录"按钮
  2. 查看所有处理过的文档
  3. 选择导出格式
  4. 点击"导出"按文档批量导出所有结果

❓ 常见问题

Q: 识别准确率不高怎么办？

A: 尝试以下方法：
• 确保文档图片清晰，分辨率足够
• 选择更适合的AI模型
• 对于复杂文档，可以尝试不同的AI服务
• 使用AI文本处理功能进一步优化结果

Q: API调用失败怎么办？

A: 请检查：
• API密钥是否正确
• 网络连接是否正常
• API服务是否有余额或配额
• 尝试切换到其他AI服务

Q: 如何提高处理速度？

A: 可以：
• 使用较小的图片分辨率
• 选择响应速度较快的AI服务
• 避免同时处理过多页面
• 利用缓存机制，避免重复处理

Q: 软件启动时提示缺少依赖怎么办？

A: 这通常是 libvips 依赖缺失导致的：
• Linux用户：运行 sudo apt-get install libvips-dev
• macOS用户：运行 brew install vips
• Windows用户：从GitHub下载libvips预编译包并配置
• 安装完成后重新启动应用

Q: PDF页面无法正常显示怎么办？

A: 这可能是图像处理问题：
• 确保 libvips 依赖已正确安装
• 检查PDF文件是否损坏
• 尝试重新启动应用
• 如果问题持续，请联系技术支持

Q: 软件是否免费？

A: 识文君软件本身个人免费使用，但AI服务可能需要付费。Pollinations AI提供免费服务，适合日常使用。
